# [Kalcyoulater](https://leonides.dev/kalcyoulater)

Smart kitchen utility for converting ingredient amounts/units and calculating calories from recipes.

## About

When working with recipes, it can be tricky to adjust measurements or calculate nutritional values when you change serving sizes, switch units, or substitute ingredients. **Kalcyoulater** makes this process effortless: just input an ingredient's amount and unit from your recipe, enter its nutrition info (serving size, calories), and instantly find out the equivalent in a new amount/unit along with the calculated calorie count.

Kalcyoulater helps cooks, bakers, meal preppers—anyone who needs precise, fast nutrition conversions right in their kitchen or on their devices.

## Features

- 🧮 Converts recipe ingredient amounts and units easily (e.g., cups to grams, tbsp to oz)
- 🔄 Flexible: input original and target units/amounts for a single ingredient
- 🍎 Calculates calories per amount using user-supplied nutritional info (serving size, calories)
- 💡 Designed for everyday kitchen use—no fuss, no confusion
- 📱 Offline mode (Progressive Web App)

## Technical Details

Statically built with [TypeScript](https://www.typescriptlang.org/) and [Vite](https://vitejs.dev/).  
CI/CD pipeline using [GitHub Actions](https://github.com/features/actions), deployed to [GitHub Pages](https://pages.github.com/).
